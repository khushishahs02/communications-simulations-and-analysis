# Communications Simulations and Analysis

A collection of MATLAB simulations and analysis work completed as part of the Communication Systems Lab. Each lab includes MATLAB `.m` files for hands-on exploration and a PDF report documenting observations and results.

---

## 🚀 Getting Started

To run the MATLAB scripts:

1. Open **MATLAB** (R2020a or later recommended)
2. Navigate to the desired lab folder
3. Open and run any `.m` or `.mlx` file

```matlab
% Example – run from MATLAB command window
cd LAB04
run('lab04_01.m')
```

---

## 📄 Reports

Each lab folder contains a PDF report with:
- Objectives and methodology
- Simulation results and plots
- Analysis and observations
- Conclusions

---

## 🛠 Requirements

- MATLAB (R2020a or later)
- Communications Toolbox *(recommended for modulation/demodulation functions)*
